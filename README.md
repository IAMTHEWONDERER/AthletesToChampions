# Athletes to Champions

![Athletes to Champions](atc_frontend/public/image.png)

**Train harder. Track smarter. Win bigger.**

Performance tracking software built for athletes who refuse to settle. From first session to the championship podium.

## Overview

This repository holds the codebase for **Athletes to Champions**, an end-to-end performance tracking suite:
- **atc_frontend**: A modern, interactive web application built with React, TypeScript, and Vite using Framer Motion for stunning animations.
- **atc_backend**: The data and business logic engine powered by Python.

## Getting Started

### Frontend 
Navigate to the frontend directory to run the UI:
```bash
cd atc_frontend
npm install
npm run dev
```

### Backend
Navigate to the backend directory and activate your environment to run the services:
```bash
cd atc_backend
# Activate your venv
.\.venv\Scripts\Activate.ps1
# Install dependencies
pip install -r requirements.txt
# Run the backend
python main.py
```
